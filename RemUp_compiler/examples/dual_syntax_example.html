什么是 RemUp

RemUp 是一个用于辅助学习记忆的轻量级标记语言 专注于知识管理 帮助用户构建知识体系 ↩ 跳回原文 你可以简单地认为这是一个专业化的 Markdown

核心特性
  • 🎴 主卡系统结构化知识承载
  • 💡 注卡系统交互式批注
  • 📚 归档系统智能知识组织
  • 🎨 多主题支持一键切换样式
传统注卡

这是传统语法的注卡： 变量 存储数据的容器 ↩ 跳回原文 编程基础概念

传统解释

行内解释使用符号，例如：Python >>一种高级编程语言

极简注卡

这是极简语法的注卡： 函数 一段可重复使用的代码 ↩ 跳回原文 提高代码复用性

极简解释

行内解释使用符号，例如：JavaScript^前端开发的主要语言
