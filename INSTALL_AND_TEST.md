# RemUp VSCode 扩展 - 安装与测试指南

## 🎯 快速安装（30 秒）

### 方法一：VSCode 内安装（推荐）

```bash
步骤：
1. 打开 VSCode
2. 按 Ctrl+Shift+X 打开扩展面板
3. 搜索 "RemUp"
4. 点击 "安装"
5. 等待安装完成
```

**预计耗时**: 30 秒  
**难度**: ⭐

### 方法二：本地打包安装

```bash
步骤：
1. 进入插件目录
   cd vscode-remup-syntax/RemUp

2. 使用 vsce 打包（需要先安装）
   npm install -g @vscode/vsce
   vsce package

3. 安装生成的 .vsix 文件
   - 在 VSCode 中按 Ctrl+Shift+P
   - 输入 "Extensions: Install from VSIX"
   - 选择生成的 .vsix 文件
```

**预计耗时**: 2 分钟  
**难度**: ⭐⭐⭐

## 🔧 故障排除

### 问题 1：没有自动换行

**可能原因**:
1. 文件扩展名不是 `.remup` 或 `.ru`
2. 语言模式不是 "RemUp"
3. 配置未生效

**解决方案**:
```bash
1. 确认文件扩展名正确
2. 点击右下角语言模式，选择 "RemUp"
3. 检查设置：remup.enableAutoWrap = true
4. 重启 VSCode
```

---

### 问题 2：语法不高亮

**可能原因**:
1. 扩展未安装
2. 文件关联错误
3. 语法文件损坏

**解决方案**:
```bash
1. 确认扩展已安装
2. 重新安装扩展
3. 检查 language-configuration.json 格式
4. 重新加载窗口 (Ctrl+Shift+P → Reload Window)
```

---

### 问题 3：配置不生效

**可能原因**:
1. JSON 格式错误
2. 配置作用域错误
3. 与其他扩展冲突

**解决方案**:
```bash
1. 检查 settings.json 格式
2. 使用 ConfigurationTarget.Workspace
3. 禁用其他可能冲突的扩展
4. 查看开发者工具日志
```
