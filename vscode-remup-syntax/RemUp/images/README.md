# 图标资源说明

## 需要的图标文件

为了让扩展完整显示，请添加以下图标文件到此目录：

### 必需图标

1. **images/icon.png** (主图标)
   - 尺寸：128x128 像素
   - 格式：PNG 透明背景
   - 用途：扩展市场展示、侧边栏图标

### 可选图标

2. **images/icon-light.svg** (浅色主题图标)
   - 格式：SVG
   - 用途：语言配置文件图标（浅色主题）

3. **images/icon-dark.svg** (深色主题图标)
   - 格式：SVG
   - 用途：语言配置文件图标（深色主题）

## 设计建议

### Logo 设计理念

- **简洁现代**: 使用简洁的几何图形
- **记忆相关**: 可以包含卡片、标签、链接等元素
- **色彩搭配**: 建议使用蓝色、紫色等知识/科技色调
- **可识别性**: 在小尺寸下仍保持清晰可辨

### 参考元素

可以从以下元素获取灵感：

- 📇 卡片/笔记形状
- 🏷️ 标签图标
- 🔗 链接符号
- 💡 灯泡（表示学习/理解）
- 🧠 大脑轮廓（表示记忆）
- 📚 书本/文档

## 临时方案

如果暂时没有图标，可以：

1. 使用 VSCode 默认图标
2. 从项目主 Logo 导出简化版本
3. 使用在线工具生成占位图标

## 导出步骤

1. 打开项目 Logo 文件 `static/Logo.svg`
2. 在 Illustrator/Figma 中简化设计
3. 导出为 128x128 PNG 和 SVG 格式
4. 保存到 `images/` 目录

---

**注意**: 当前版本没有配置图标也不会影响语法高亮功能，只是扩展面板中显示默认图标。
