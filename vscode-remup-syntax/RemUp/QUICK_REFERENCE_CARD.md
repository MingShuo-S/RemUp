# RemUp 语法速查卡片

## 🎯 一分钟上手

### Markdown 风格（推荐 ⭐）

```remup
# 归档名
## 卡片主题
(!: 重要) (>: #参考)

### 区域名
内容 ^简短解释

`术语`[批注] 或 [内容 | 批注]

/+>
```

### 传统风格

```remup
--<归档>--
<+卡片
(!: 重要)

---区域
内容 >>解释

`术语`[批注]

/+>
```

## 📋 完整语法对照表

| 功能 | Markdown 风格 | 传统风格 | 说明 |
|------|-------------|---------|------|
| **归档** | `# 名称` | `--<名称>--` | 知识分组 |
| **卡片** | `## 主题` | `<+主题` | 知识单元 |
| **结束** | 自动/`/+>` | `/+>` | 结束卡片 |
| **区域** | `### 名称` | `---名称` | 内容分区 |
| **次级** | `#### 主题` | ❌ | 细粒度分组 |
| **重要** | `(!: 内容)` | `(!: 内容)` | 红色高亮 |
| **链接** | `(>: #目标)` | `(>: #目标)` | 蓝色跳转 |
| **评级** | `(*: ⭐⭐⭐)` | `(*: ⭐⭐⭐)` | 星级评价 |
| **注卡** | `[内容 \| 批注]` | `` `内容`[批注] `` | 悬停显示 |
| **解释** | `^文字` | `>>文字` | 灰色补充 |
| **代码** | `\`\`\`语言` | `\`\`\`语言` | 代码块 |
| **列表** | `-` 或 `1.` | `-` 或 `1.` | 项目列表 |
| **粗体** | `**文字**` | `**文字**` | 加粗显示 |

---

## 🏷️ 标签符号详解

| 符号 | 含义 | 颜色 | 示例 |
|------|------|------|------|
| `!` | 重要/警告 | 🔴 红色 | `(!: 必考)` |
| `>` | 引用/跳转 | 🔗 蓝色 | `(>: #前置)` |
| `*` | 评级/重点 | ⭐ 金色 | `(*: ⭐⭐⭐)` |
| 其他 | 备注/说明 | ⚪ 灰色 | `(难度：高)` |

## 💻 VSCode 代码片段

### 结构类
- `#archive` → `# 归档名称`
- `##card` → `## 卡片主题` + 完整结构
- `###region` → `### 区域名`
- `####subcard` → `#### 次级卡片`
- `--archive` → `--<名称>--`
- `<+card` → `<+主题 ... /+>`

### 元素类
- `vibe` → `` `内容`[批注] ``
- `[vibe` → `[内容 | 批注]`
- `>>exp` → `>>解释`
- `^exp` → `^解释`
- `!label` → `(!: 内容)`
- `>label` → `(>: #目标)`
- `*label` → `(*: ⭐⭐⭐)`

### 模板类
- `` ``` `` → 代码块模板
- `fullcard` → 完整卡片模板
- `note` → 学习笔记模板

## 📝 实用模板

### 学习笔记模板

```remup
# 课程名称

(!: 重点) (*: ⭐⭐⭐)

## 核心概念 1
(!: 基础)

### 定义
关键概念 ^关键词

### 示例
- 例子 `术语`[说明]
- 例子 [内容 | 批注]

## 核心概念 2
(!: 进阶) (前置:#概念 1)

### 应用
实际场景 ^理解

/+>
```

### 复习卡片模板

```remup
# 复习专题

## 知识点 A
(!: 必考) (*: ⭐⭐⭐)

### 要点
- 要点 1 ^关键词
- 要点 2 ^关键词

### 对比
vs`[A]`^区别
vs`[B]`^联系

/+>
```

## ⚡ 快捷键

### VSCode 通用
- `Ctrl+Space` - 触发建议（包含代码片段）
- `Ctrl+K Ctrl+F` - 折叠选中区域
- `Ctrl+K Ctrl+J` - 展开所有折叠

### 自动闭合
- 输入 `<+` 自动添加 `/+>`
- 输入 `` ` `` 自动配对
- 输入 `(` 自动添加 `)`
- 输入 `[` 自动添加 `]`
- 输入 `\`\`\`` 自动添加结束标记

## 🐛 快速排错

### 问题：不高亮
**解决**：检查文件扩展名是否为 `.remup` 或 `.ru`

### 问题：代码片段不显示
**解决**：按 `Ctrl+Space` 手动触发，确保在正确的文件类型中

### 问题：自动闭合无效
**解决**：检查 VSCode 设置中的 `editor.autoClosingBrackets`
