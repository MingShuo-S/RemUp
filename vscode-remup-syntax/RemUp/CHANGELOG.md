# 更新日志

本文件记录 RemUp VSCode 扩展的所有重要更新。

## [1.1.0] - 2026-03-25

### ✨ 重大更新：一体化配置

#### 🎯 核心特性
- ✅ **自动换行功能** - 默认启用，类似 Markdown 的舒适体验
- ✅ **可视化配置** - 通过设置界面即可调整所有选项
- ✅ **智能激活** - 按需加载，不影响其他语言

#### ⚙️ 新增配置项
新增 2 个可配置选项，所有设置只对 RemUp 文件生效：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `remup.enableAutoWrap` | `true` | 为 RemUp 文件启用自动换行 |
| `remup.wordWrapColumn` | `120` | 自动换行的列宽限制 |

#### 🔧 技术改进
- 📦 添加 `extension.js` 入口文件，实现配置管理逻辑
- 🎨 优化 `package.json`，添加 `contributes.configuration` 贡献点
- 🔄 监听配置变化，动态应用设置
- 📝 更新 `language-configuration.json`，移除注释以符合 JSON 规范

#### 📖 文档更新
- 📄 完善 README.md，突出 v1.1.0 新特性
- 📚 新增配置指南和快速开始文档
- 🧪 新增功能测试文件

### 🐛 Bug 修复
- 修复了 language-configuration.json 包含注释导致的解析问题
- 修复了部分用户需要手动配置才能启用自动换行的问题

## [1.0.0] - 2026-03-19

### ✨ 新增功能

#### 双语法支持
- ✅ 完整支持 Markdown 风格语法（`#`, `##`, `###`, `####`）
- ✅ 完整支持传统语法（`--<`, `<+`, `---`, `/+>`）
- ✅ 自动识别两种语法模式并正确高亮

#### 语法高亮增强
- 🏷️ 标签系统高亮优化（支持中英文冒号）
- 💬 注卡语法高亮（传统 `` `内容`[批注] `` 和 Markdown `[内容 | 批注]`）
- 📝 行内解释高亮（传统 `>>` 和 Markdown `^`）
- 🎯 次级卡片 `####` 高亮支持

#### 智能编辑功能
- 🔄 自动括号闭合（`<+` → `/+>`，`` ` `` → `` ` ``）
- 📦 代码块自动闭合（`\`\`\``）
- 🗂️ 语法折叠支持（按归档、卡片、区域层级折叠）
- ⚡ 50+ 个代码片段模板

#### 代码片段库
- `#archive`, `##card`, `###region`, `####subcard` - Markdown 结构
- `--archive`, `<+card`, `---region` - 传统结构
- `vibe`, `[vibe` - 注卡快捷输入
- `>>exp`, `^exp` - 解释快捷输入
- `!label`, `>label`, `*label` - 标签快捷输入
- `fullcard`, `note` - 完整模板

### 🔧 技术改进
- 📄 修复 file extensions 配置（`.remup` 和 `.ru` 分开配置）
- 🎨 优化语言配置文件（注释、括号、折叠规则）
- 📚 完善 package.json 元数据（关键词、仓库链接、分类）
- 🏷️ 统一 scopeName 为 `source.remup`

### 📖 文档更新
- 📝 完整的 README 文档（中英双语支持）
- 📋 详细的使用说明和示例
- 🎮 代码片段使用指南
- 🔧 故障排除章节

### 🐛 Bug 修复
- 修复了标签正则表达式不支持中文冒号的问题
- 修复了卡片结束检测逻辑
- 修复了代码块嵌套处理问题
